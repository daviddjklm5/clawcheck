本文档用于记录 `automation/sql` 下 SQL 文件的编号、用途、关联方案和当前状态。

## 4. 执行边界

- `验收` 类型 SQL 不应作为建表迁移重复执行，除非发布步骤明确要求。
- `历史补丁` 保留用于老库补齐或追溯；新库初始化应优先走最新建表 SQL 和代码内补齐逻辑。
- `回滚` SQL 只能在明确回滚窗口内执行，不得作为日常修复脚本使用。
